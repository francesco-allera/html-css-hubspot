# EXERCISE 9 - BOOLEAN CAREERS TRAINEE:


### Languages used:

- HTML5
- CSS3


### Purpose of the exercise:

Exercise done using HTML and CSS. Practice with HTML syntax and the different types of tags (block, inline, and self closing). Reproducing the layout of the HubSpot landing page.

Exercising the CSS reset,  similar style and classes for different items, position relative and absolute, customizing buttons, use of the :hover property, practicing various background properties, use of clearfix and floats, and use of ::after for creating dynamics items on page.
